ATAJOS DE TECLADO

Los atajos de teclado deben ir en C:\Users\\AppData\Roaming\Code\User\keybindings.json

Estos atajos sobrescribirán los que por defecto tenga VS Code

[ { "key": "ctrl+shift+oem_102", // para poner un tag, clase u otra cosa a un texto seleccionado "command": "editor.emmet.action.wrapWithAbbreviation" }, { "key": "ctrl+down", // para cambiar el foco a la terminal "command": "workbench.action.terminal.focus", }, { "key": "ctrl+up", // para cambiar el foco al editor (archivo que se está editando) "command": "workbench.action.focusActiveEditorGroup", "when": "terminalFocus" } ]

Para ver los archivos ocultos en VS Code (carpetas .git por ejemplo).

1. Ir a file > Preferences > Settings (o Ctrl + ,) 2. En la columna de la izquierda seleccionar: User > Commonly used > Text editor > Files 3. En el panel de la derecha buscar el item 'Exclude' 4. Posar el cursor sobre la opción que dice **/.git y hacia la derecha hacel clic en la X para eliminar este item. Se le está diciendo a VS Code que NO se quiere ocultar las carpetas '.git' Con esto la carpeta .git debería aparecer en la lista de archivos/carpetas de VS Code disponibles para trabajar.

Para usar Emmet: Wrap with abbreviation

1. Seleccionar un texto. 2. teclear Ctrl + Shift + p 3. teclaear 'wrap' para que aparezca la opción: Emmet: Wrap with abbreviation 4. hacerle clic y teclear la etiqueta que envolverá el texto selecionado 5. presionar Enter para terminar También se puede envolver con una clase al texto seleccionado Video referencia: https://www.youtube.com/shorts/ytL1aaXjkxU
